# 異常值（Outlier）

以法務部調查局北部地區機動工作站肅貪組為主軸，描寫科技公司、地方政府與政商網絡交織而成的長篇犯罪調查小說。

## 故事核心

北機站調查官殷皓明與肅貪組追查西利康電通及其關係企業，逐步揭開劣質科技產品、公共交通控制系統、官商勾結與利益輸送背後的完整網絡。

## 主要勢力

- 法務部調查局北部地區機動工作站
- 新北市政府政風處
- 新北市政府交通局交通控制中心
- 臺北市政府警察局及相關分局
- 西利康電通
- 盛御半導體與盛昱電子
